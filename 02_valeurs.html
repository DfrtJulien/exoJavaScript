Valeurs

Observez le programme valeurs.js et tentez de prévoir les valeurs finales de chaque variable.

var a = 2;

valeur = 2

a = a - 1;

valeur = 1

a++;

valeur de a = 2

var b = 8;

b += 2;

valeur prévu b = 10

var c = a + b * b;

valeur de c = 102

var d = a * b + b;

valeur prévu 30

var e = a * (b + b);

valeur prévu 40

var f = a * b / a;

valeur prévu = 10

var g = b / a * a;

valeur prévu 10

Vérifiez vos prévisions en créant ce programme puis en le déboguant pour observer les valeurs des variables.
